# Prompt 构建优化任务

## 问题分析
用户输入："蓝色的天空飞着一条龙"
生成结果：与 prompt 无关的图片

## 根本原因
prompt-builder.js 中添加了太多额外约束和说明，导致 Doubao API 无法正确理解用户的核心需求。

## 解决方案

### 1. 简化 prompt 构建逻辑
- 直接使用用户的 customPrompt 作为核心描述
- 减少额外添加的约束
- 确保"龙"等词汇不被误判

### 2. 优化敏感词过滤
- 检查"龙"是否被误判
- 添加白名单机制（常见动物/神话生物）
- 减少过度过滤

### 3. 改进 API 调用
- 直接使用用户 prompt 调用 Doubao
- 减少中间处理层
- 添加调试日志

## 需要修改的文件
- utils/prompt-builder.js - 简化构建逻辑
- utils/sensitive-filter.js - 添加白名单
- api/generate.js - 优化 API 调用

## 验收标准
- 用户输入"蓝色的天空飞着一条龙"能正确生成龙
- 敏感词过滤仍然有效
- 构建验证通过
- 测试通过
