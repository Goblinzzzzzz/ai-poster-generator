# 任务：输入框优化 - 方案 A

## 任务描述
保持当前底部固定紧凑风格，添加引导文案、优化占位符、添加简化版功能入口。

## 背景
对标即梦官网，当前输入框缺少：
1. 引导文案
2. 功能入口卡片
3. 占位符文案可优化

## 设计要求

### 1. 添加引导文案（输入框上方）
```
开启你的 AI 创作之旅 ✨
描述你想生成的画面，AI 将帮你生成精美海报
```

**样式要求：**
- 居中显示
- 字体：14-16px
- 颜色：主标题深色，副标题灰色
- 与输入框间距：12-16px

### 2. 优化占位符
```
描述主体、材质、光线、配色和构图，例如：高端护肤产品海报，玻璃器皿，柔和晨光，冷白配色，极简留白...
```

### 3. 添加功能入口（输入框下方，可选）
```
[🎨 图片生成] [🎬 视频生成] [✏️ 智能画布]
```

**样式要求：**
- 3 个按钮水平排列
- 图标 + 文字
- 点击跳转到对应功能（暂时可先实现图片生成）
- 与输入框间距：12px
