# P2 阶段 - 用户体验优化任务

## 任务概述
优化生成进度提示、错误提示，并删除前端假数据。

---

## P2-001: 生成进度优化

### 需求
1. 生成过程中显示进度状态
2. 支持生成过程中也正常显示卡片
3. 图片位置显示"生成中..."状态
4. 生成完成后自动刷新显示真实图片

### 实现方案
- 添加"生成中"状态卡片
- 卡片正常显示（作者/提示词/操作按钮）
- 图片位置显示加载动画 + "正在生成中..."
- 生成完成后替换为真实图片

### 文件
- src/components/GeneratingCard.jsx (新建)
- src/components/GeneratingCard.css (新建)
- src/App.jsx (修改 - 集成生成中状态)

---

## P2-002: 完善错误提示

### 需求
1. 生成失败时显示明确错误原因
2. 敏感词被过滤时提示用户修改
3. 网络错误时提供重试按钮
4. 错误提示友好且可操作

### 错误类型
- **敏感词错误** - "输入内容包含敏感词汇，请修改后重试"
- **网络错误** - "网络连接失败，请检查网络后重试"
- **API 错误** - "生成服务暂时不可用，请稍后重试"
- **其他错误** - 显示具体错误信息

### 文件
- src/components/ErrorAlert.jsx (新建/修改)
- src/components/ErrorAlert.css (新建)
- src/App.jsx (修改 - 改进错误处理)
- src/components/PromptInput.jsx (修改 - 显示错误)

---

## P2-003: 删除前端假数据

### 需求
1. 删除 App.jsx 中的 SEED_WORKS 假数据
2. 时间线为空时显示空状态引导
3. 空状态显示：
   - "还没有作品"
   - "开始你的第一个创作"
   - 提示词建议

### 文件
- src/App.jsx (修改 - 删除 SEED_WORKS)
- src/components/TimelineFeed.jsx (修改 - 空状态优化)
