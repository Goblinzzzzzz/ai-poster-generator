# 任务：顶部导航栏改造成即梦风格

## 任务描述
将当前项目的顶部导航栏（TopBar）改造成即梦风格，更简洁、更紧凑。

## 背景
用户反馈当前顶部导航栏太大、太复杂，希望改成即梦官网的简洁风格。

## 设计要求

### 最终布局
```
┌─────────────────────────────────────────────────────────────────┐
│  ☰  生成                              🔍  📅  [LU▼]            │
└─────────────────────────────────────────────────────────────────┘
高度：50px（原 80-90px）
```

### 左侧
- 汉堡菜单（移动端显示）
- 视图标题 + 描述（缩小）

### 右侧
- 搜索图标（点击展开搜索框）
- 时间图标（点击展开时间选项）
- 用户按钮（保留）

### 关键特性
- **高度**：50px（移动端 56px）
- **背景**：默认透明 → 悬停/激活时半透明毛玻璃
- **搜索**：图标式，点击展开 240px 搜索框
- **时间**：图标式，点击展开时间选项列表
- **交互**：失焦自动关闭下拉菜单
