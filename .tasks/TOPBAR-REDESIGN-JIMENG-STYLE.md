## 🔧 修改内容

### 1. TopBar.jsx - 组件重构
**删除的 Props：**
- ❌ `mediaFilter` / `onMediaFilterChange` / `mediaOptions`
- ❌ `actionFilter` / `onActionFilterChange` / `actionOptions`
- ❌ `resultCount`

**新增的功能：**
- ✅ 搜索图标点击展开搜索框
- ✅ 时间图标点击展开时间选项
- ✅ 失焦自动关闭下拉菜单

**新增的 Icons：**
- `CalendarIcon` - 日历图标（时间筛选）

### 2. TopBar.css - 样式重写
**尺寸调整：**
| 属性 | 原值 | 新值 |
|------|------|------|
| 高度 | 80-90px | 50px |
| Padding | 18px 24px | 8px 20px |
| 图标按钮 | 46x46px | 36x36px |
| 用户头像 | 32x32px | 28x28px |

**背景效果：**
```css
/* 默认：透明 */
background: transparent;
backdrop-filter: blur(0);

/* 悬停/激活：半透明毛玻璃 */
.topbar.has-active-dropdown,
.topbar:hover {
  background: rgba(247, 249, 252, 0.95);
  backdrop-filter: blur(14px);
}
```

**下拉菜单样式：**
- 搜索框：240px 宽，圆角 12px
- 时间选项：140px 宽，垂直列表
- 背景：白色半透明 + 毛玻璃
- 阴影：柔和的多层阴影

### 3. App.jsx - 简化 Props
删除不需要的筛选器 props，只保留：
- `searchValue` / `onSearchChange`
- `timeFilter` / `onTimeFilterChange` / `timeOptions`
