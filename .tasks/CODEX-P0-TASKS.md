## P0-001: 极简创作模式

### 需求

### 具体要求
1. 创建居中容器（max-width: 900px, margin: 0 auto）
2. 实现大文本框（min-height: 200px, placeholder: "描述你想生成的画面..."）
3. 大尺寸生成按钮（width: 100%, padding: 1.5rem, 渐变背景）
4. 默认隐藏高级参数（只显示核心输入）

## P0-002: 历史记录功能

### 需求
实现本地存储和展示历史作品。

### 具体要求
1. localStorage 存储生成的历史（最多 50 条）
2. 瀑布流展示历史作品
3. 每条记录包含：图片、提示词、时间
4. 点击作品可查看详情
5. 支持"再次生成"

### 文件创建
- src/utils/storage.js - 存储工具
- src/components/HistoryGallery.jsx - 历史画廊组件
- src/components/WorkDetail.jsx - 作品详情组件
