# 任务：完全对标即梦网页版 - 图片生成界面架构重构

## 任务背景
当前我们的 AI Poster Generator 前端界面采用"垂直堆叠表单"设计，用户体验像"填表"，不符合现代 AI 图片生成工具的交互逻辑。

即梦（Jimeng）网页版采用"左右分栏 + 底部展示"架构，用户体验是"创作"而非"填表"。

## 目标
完全重构前端架构，对标即梦网页版的交互逻辑和视觉设计。

## 核心需求

### 1. 布局架构变革

**当前布局（需要废弃）：**
- 垂直堆叠表单
- 字段包括：海报类型、尺寸模板、标题、副标题、风格、Logo 上传、参考图上传
- 用户体验：填表式

**目标布局（即梦架构）：**
```
┌────────────────────────────────────────────┐
│  顶部导航栏 (Logo | 灵感 | 生成 | 资产)      │
├────────────────────────────────────────────┤
│                                            │
│  ┌─────────────┐  ┌──────────────────┐     │
│  │             │  │                  │     │

│  │  提示词     │  │  配置面板        │     │
│  │  输入区     │  │  (固定宽度)      │     │
│  │  (大文本框) │  │                  │     │
│  │             │  │  - 比例选择      │     │
│  │  [模板]     │  │  - 风格选择      │     │
│  │  [参考图]   │  │  - 高级设置      │     │

│  └─────┘ └─────┘ └─────┘ └─────┘          │
```

### 2. 左侧提示词输入区

**功能要求：**
- 大文本框，高度至少 180px（6 行）
- 占位符文案："例如：一位穿着现代职业装的女性，办公室环境，极简主义风格，柔和自然光，三分法构图..."
- 提示词模板快速填充（4-6 个预设）
- 参考图上传区域（拖拽样式）
- 生成按钮（渐变背景，大尺寸）

**代码示例：**
```jsx
<textarea
  placeholder="例如：一位穿着现代职业装的女性，办公室环境，极简主义风格，柔和自然光，三分法构图..."
  value={prompt}
  onChange={(e) => setPrompt(e.target.value)}
  rows={6}
/>

{/* 提示词模板 */}
<div className="template-chips">
  {TEMPLATES.map(t => (
    <button onClick={() => setPrompt(t.text)}>
      {t.text.slice(0, 20)}...
    </button>
  ))}
</div>

{/* 参考图上传 */}
<label className="upload-area">
  <input type="file" accept="image/*" />
  <span>📎 添加参考图</span>
</label>

{/* 生成按钮 */}
<button className="generate-btn">
  ✨ 立即生成
</button>
```

### 4. 底部作品展示区

**功能要求：**
- 瀑布流网格布局
- 作品卡片（图片 + 操作按钮）
- 悬停动画效果
- 操作按钮：下载、同款

**代码示例：**
```jsx
<div className="gallery-grid">
  {images.map(img => (
    <div className="gallery-item">
      <img src={img} />
      <div className="actions">
        <button>⬇️ 下载</button>
        <button>🔄 同款</button>
      </div>
    </div>
  ))}
</div>
```

### 5. 视觉设计要求

**即梦风格元素：**
- 渐变按钮：`linear-gradient(135deg, #2563eb, #7c3aed)`
- 卡片阴影：`0 4px 20px rgba(0, 0, 0, 0.08)`
- 圆角设计：`border-radius: 12-16px`
- 选中状态：渐变背景 + 蓝色边框
- 悬停动画：`transform: translateY(-2px)`

**颜色方案：**
- 主色调：`#2563eb` (蓝色)
- 渐变色：`#7c3aed` (紫色)
- 背景色：`#f8fafc` (浅灰)
- 文字色：`#0f172a` (深蓝)

### 6. 响应式设计要求

**桌面端（>1024px）：**
- 左右分栏布局
- 右侧面板固定 400px 宽度

**平板端（640-1024px）：**
- 单列布局
- 上下堆叠

**移动端（<640px）：**
- 全宽布局
- 优化触摸体验
