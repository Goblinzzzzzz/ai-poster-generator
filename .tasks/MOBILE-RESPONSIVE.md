# 移动端适配优化任务

## 问题分析
当前 Sidebar 在移动端 (< 860px) 被隐藏，但没有替代导航方案。

## 需要实现

### 1. 移动端底部导航栏
- 固定在底部
- 显示核心功能图标 (灵感/生成/资产)
- 当前选中状态高亮
- 高度：60px

### 2. 汉堡菜单 (可选)
- 左上角汉堡按钮
- 点击展开侧边抽屉
- 显示完整导航项

### 3. 布局优化
- 内容区自适应
- 触摸友好 (按钮大小 ≥ 44px)
- 字体大小适配
